Squad 2 · Backend e Segurança
Proteção de dados
por design — não por decreto.
Cinco módulos Python que garantem que nenhum dado pessoal chegue à IA, que arquivos sejam criptografados antes de tocar o disco, e que cada ação seja rastreada sem expor o usuário.

LGPD conforme
🔒
Anonimização Automática (RCP-001)
CPF, CNPJ, RG, e-mail, telefone e nomes próprios substituídos automaticamente antes de qualquer envio à IA — o modelo nunca vê dados reais.
🔐
Criptografia AES-128 (RNF-010)
Todos os arquivos PDF são criptografados com Fernet antes de tocar o disco. Chave carregada de variável de ambiente, nunca no código.
📋
Logs de Auditoria (RNF-012)
Cada ação registrada com timestamp UTC. IP armazenado como hash SHA-256 — rastreável tecnicamente, irreversível pessoalmente.
🛡️
Opt-Out de IA (RCP-002)
Usuário controla se seus dados podem treinar modelos de IA. Proteção máxima por padrão — opt_out = true se usuário não encontrado.
Anonimização de Dados
Como o dado pessoal é removido antes de chegar à IA
ℹ️ O módulo anonimizador.py processa o texto antes de qualquer armazenamento ou envio. A proteção é estrutural — acontece automaticamente, sem ação do usuário.
